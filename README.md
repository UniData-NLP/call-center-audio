# Call Center  Dataset - 13,000+ Hours
 Dataset is a large audio dataset containing **13,000+** hours of real-world customer service calls from global call centers, featuring **90%+ unique speakers** and **time-stamped transcripts** for accurate **speech recognition**, **speaker diarization**, and **conversational AI model** training. - **[Get the data](https://unidata.pro/datasets/call-center-audio/?utm_source=github-nlp&utm_medium=referral&utm_campaign=call-center-audio)**
## Dataset characteristics:
| Characteristic          | Data                                                                 |
|-------------------------|----------------------------------------------------------------------|
| Description             | Audio of real customer service calls                                 |
| Data types              | Audio                                                                |
| Tasks                   | Speech Recognition, Speaker Diarization                             |
| Hours of audio          | 13,000+                                                              |
| Language                | English: 96%, Spanish: 2.5%, Hindi: 1%, Other languages: 0.5%      |
| Labeling                | Metadata (id, company, category, device, OS, city, state, country, duration, wait time, transcription, AI summary) |
